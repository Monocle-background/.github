# Monocle — Screen Noise Cancellation: Blur Background Windows

## About Monocle

Monocle provides real-time blur for background windows, keeping only your active window in sharp focus. Monocle eliminates visual clutter and reduces screen distractions. Monocle combines adjustable blur intensity, exclusion lists for specific apps, and smooth focus transitions. Monocle enables users to concentrate deeply on single tasks without closing or minimizing other windows. Monocle works alongside mission control, spaces, and window managers. Monocle is trusted by writers, programmers, researchers, data analysts, video editors, and knowledge workers worldwide.

## Quick Reference for Monocle

| Operation | Monocle Action |
|-----------|-----------------|
| Toggle blur on/off | Ctrl+Shift+B |

| Add focused app to exclusions | Ctrl+Shift+E |

| Open settings panel | Ctrl+Shift+S |
| Reload configuration | Ctrl+Shift+R |
| Reset to defaults | Ctrl+Shift+D |
| Show current status | Ctrl+Shift+I |
| Pause Monocle temporarily | Ctrl+Shift+P |
